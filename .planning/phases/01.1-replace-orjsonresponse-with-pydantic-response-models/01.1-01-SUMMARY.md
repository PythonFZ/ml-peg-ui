## Decisions Made
- Used return type annotations (`-> HealthResponse`) rather than `response_model=` kwarg — the latter would strip extra fields from BenchmarkRow, defeating the `extra="allow"` config
- Kept `models_cache` as `list[dict]` in `app.state` — `ModelEntry.model_validate()` called on return, not on cache write (cache format unchanged, simple)
- `envelope()` helper removed entirely — `Meta` model provides the same contract, type-safely
