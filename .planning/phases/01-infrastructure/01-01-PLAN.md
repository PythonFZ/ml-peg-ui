2. Add concurrently as dev dependency: `bun add --dev concurrently`

3. Set up `package.json` scripts:
   ```json
   {
     "scripts": {
       "dev": "concurrently --names 'next,api' --prefix-colors 'blue,green' 'next dev' 'uv run uvicorn api.index:app --reload --port 8000'",
       "build": "next build",
       "start": "next start"
     }
   }
   ```

4. Create `next.config.ts` with local dev rewrite:
   ```typescript
   import type { NextConfig } from "next";

   const nextConfig: NextConfig = {
     async rewrites() {
       return [
         {
           source: "/api/:path*",
           destination: "http://127.0.0.1:8000/api/:path*",
         },
       ];
     },
   };

   export default nextConfig;
   ```

5. Create `src/app/layout.tsx` — minimal root layout with html/body tags, metadata title "ML-PEG Leaderboard".

6. Create `src/app/page.tsx` — placeholder page with heading "ML-PEG Leaderboard" and brief description text. Keep it simple — this is a placeholder. Do NOT use fancy styling or the frontend-design skill; this is infrastructure scaffolding.

7. Create `src/lib/types.ts` with types derived from actual data shape (per CONTEXT.md and RESEARCH.md):
   ```typescript

     data: T;
     meta: { count?: number; columns?: string[] };

   export interface Category {
     slug: string;

     Score: number;
   ```
