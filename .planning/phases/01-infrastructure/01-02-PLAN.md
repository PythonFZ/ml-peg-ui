    - FilesystemBackend.get_json reads JSON from data/{path} using orjson
    - FilesystemBackend.list_keys lists files/dirs under data/{prefix}

    - GET /api/v1/health returns {"status": "ok"}
    - GET /api/v1/categories returns {"data": [...categories with embedded benchmarks...], "meta": {"count": N}}
    - GET /api/v1/benchmarks/{slug}/table returns {"data": [...rows...], "meta": {"count": N, "columns": [...]}}
    - GET /api/v1/models returns {"data": [...models...], "meta": {"count": N}}
    - Unknown benchmark slug returns 404
  </behavior>
  <action>
1. Create `api/__init__.py` (empty).

2. Create `api/storage.py`:
   - Define `StorageBackend` as a `typing.Protocol` with methods: `get_json(path: str) -> dict`, `list_keys(prefix: str) -> list[str]`, `presigned_url(path: str) -> str`.
   - `FilesystemBackend`:
     - `__init__(self, base_path: str = "data")` — stores `pathlib.Path(base_path)`
     - `get_json`: reads file at `base_path / path`, parses with `orjson.loads`
     - `list_keys`: lists directories/files under `base_path / prefix`, returns relative paths
     - `presigned_url`: raises `NotImplementedError("Presigned URLs not available in filesystem mode")`
   - `MinioBackend`:
     - `__init__(self)` — creates `minio.Minio` client from env vars (MINIO_ENDPOINT, MINIO_ACCESS_KEY, MINIO_SECRET_KEY). Stores `self.bucket` from MINIO_BUCKET and `self.prefix` from MINIO_PREFIX (default "").
     - `get_json`: calls `self.client.get_object(self.bucket, self._key(path))`, parses response bytes with `orjson.loads`
     - `list_keys`: calls `self.client.list_objects(self.bucket, prefix=self._key(prefix))`, returns keys
     - `presigned_url`: calls `self.client.presigned_get_object(self.bucket, self._key(path), expires=timedelta(hours=1))`
     - `_key(path)`: returns `f"{self.prefix}/{path}"` if prefix else `path`
   - `create_storage() -> StorageBackend`: returns `MinioBackend()` if `os.environ.get("MINIO_ENDPOINT")` else `FilesystemBackend()`

3. Create `api/index.py`:
   - Load `.env.local` with `load_dotenv(".env.local")` at module top (no-op if absent)
   - Build a **category/benchmark index** at startup by scanning the storage backend:
     - Scan top-level keys under `data/` (or bucket root) to discover categories
     - Skip `assets` directory
     - For each category, list benchmarks (subdirectories)
     - Build a `slug -> actual_path` mapping to handle case sensitivity (e.g., `37conf8` -> `conformers/37Conf8`)
     - Cache this index in `app.state`
   - Use `APIRouter(prefix="/api/v1")` per research guidance
   - Use `ORJSONResponse` as default response class
   - Endpoints:
     a. `GET /health` — returns `{"status": "ok"}`
     b. `GET /categories` — returns the cached category index, formatted as envelope: `{"data": [{"slug": "conformers", "name": "Conformers", "benchmarks": [{"slug": "37conf8", "name": "37Conf8"}, ...]}], "meta": {"count": 14}}`
        - Category name: title-case the directory name, replace underscores with spaces
        - Benchmark name: use the actual directory name as-is (preserves original casing)
        - Benchmark slug: lowercase version of directory name
     c. `GET /benchmarks/{slug}/table` — looks up slug in index, reads `{path}/{slug}_metrics_table.json` via storage, returns as envelope with `meta.columns` listing all column names from the first row
        - Return 404 if slug not found in index
     d. `GET /models` — scan all metrics tables (cached), extract unique (id, MLIP) pairs, return as envelope with `{"data": [{"id": "...", "display_name": "..."}], "meta": {"count": N}}`
        - Cache after first computation in `app.state.models`
   - Use FastAPI `lifespan` context manager (not deprecated `@on_event`):
     - Create storage backend via `create_storage()`
     - Build category/benchmark index
     - Store in `app.state.storage`, `app.state.categories_index`, `app.state.slug_map`
   - `envelope()` helper function: `def envelope(data, **meta): return {"data": data, "meta": {"count": len(data) if isinstance(data, list) else 1, **meta}}`
   - Add `Cache-Control: s-maxage=3600, stale-while-revalidate=86400` header to categories, table, and models endpoints

NOTE: The figure endpoint (`GET /benchmarks/{slug}/figures/{figure_slug}`) is defined in the CONTEXT.md decisions but can be stubbed for now — full implementation requires the size-based decision logic which depends on real figure file sizes. Add a stub that returns 501 with a message, so the route exists but doesn't break anything.
