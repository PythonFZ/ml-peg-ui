    - GET /api/v1/diatomics/curves/H-H returns {"data": [{"model": "...", "pair": "H-H", "distance": [...], "energy": [...]}, ...], "meta": {"count": N}}
    - GET /api/v1/diatomics/curves/Zz-Zz returns 404
    - GET /api/v1/structures/{benchmark_slug}/{model}/{filename} returns {"data": {"xyz_string": "...", "has_pbc": true}}
    - GET /api/v1/structures/{benchmark_slug}/{model}/nonexistent.xyz returns 404
    - GET /api/v1/nebs/li_diffusion/{model}/{band}/frames returns {"data": [{"energy": ..., "species": [...], "positions": [...], "lattice": [...]}, ...], "meta": {"count": 13}}
    - GET /api/v1/nebs/li_diffusion/nonexistent/b/frames returns 404
    - All endpoints set Cache-Control header
  </behavior>
  <action>
    1. Import new models in `api/index.py`: DiatomicIndexResponse, DiatomicCurve, DiatomicCurvesResponse, StructureData, StructureResponse, NebFrame, NebFramesResponse.

    2. **Diatomic index endpoint** `GET /api/v1/diatomics/index`:
       - Read `physicality/diatomics/diatomic_index.json` via `storage.get_json()`.
       - Return DiatomicIndexResponse with data=index_dict, meta=Meta(count=len(index_dict)).
       - Cache in `app.state.diatomic_index` after first load (like models_cache pattern).
       - Set Cache-Control header.

    3. **Diatomic curves endpoint** `GET /api/v1/diatomics/curves/{pair}`:
       - Read diatomic index (from app.state or load).
       - If pair not in index, return 404.
       - For each model in index[pair], read `physicality/diatomics/curves/{model}/{pair}.json` via `storage.get_json()`.
       - Build DiatomicCurve objects (model=model_id, pair from JSON, distance from JSON, energy from JSON).
       - Return DiatomicCurvesResponse. Set Cache-Control.

    4. **Structure endpoint** `GET /api/v1/structures/{benchmark_slug}/{model}/{filename}`:
       - Look up benchmark_slug in slug_map to get bench_path.
       - Read raw bytes via `storage.get_bytes(f"{bench_path}/{model}/{filename}")`.
       - Parse has_pbc: check if `pbc="T T T"` appears in the xyz content (second line).
       - Return StructureResponse with xyz_string=decoded_bytes, has_pbc=bool.
       - 404 if benchmark or file not found. Set Cache-Control.

    5. **NEB frames endpoint** `GET /api/v1/nebs/{benchmark}/{model}/{band}/frames`:
       - List files in `nebs/{benchmark}/{model}/` via `storage.list_keys()`.
       - Find file matching pattern `*-{band}-neb-band.extxyz` (handles underscore variants like `mattersim_5M`).
       - If no match, 404.
       - Read raw bytes via `storage.get_bytes()`.
       - Parse with `ase.io.read(io.StringIO(raw.decode()), index=':', format='extxyz')`.
       - Build NebFrame list: energy from `atoms.info.get("energy") or atoms.info.get("free_energy", 0.0)`, species from `get_chemical_symbols()`, positions from `atoms.positions.tolist()`, lattice from `atoms.cell.tolist() if any(atoms.pbc) else None`.
       - Return NebFramesResponse. Set Cache-Control.

    6. **Tests** — create 3 test files using httpx AsyncClient + TestClient pattern from existing tests:
       - `tests/test_diatomics.py`: test_diatomic_index (200, has data dict), test_diatomic_curves_valid (200, has curves list), test_diatomic_curves_invalid_pair (404).
       - `tests/test_structures.py`: test_structure_endpoint (200, has xyz_string and has_pbc), test_structure_not_found (404).
       - `tests/test_nebs.py`: test_neb_frames_count (200, 13 frames), test_neb_frame_schema (each frame has energy, species, positions, lattice keys), test_neb_not_found (404).
       - All tests use local `data/` directory (FilesystemBackend). Pick real model/pair/file names from actual data.
