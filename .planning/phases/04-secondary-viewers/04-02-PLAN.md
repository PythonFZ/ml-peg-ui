  <files>src/lib/types.ts, src/lib/api.ts, src/components/PeriodicTable.tsx</files>
  <action>
    1. **Add types to `src/lib/types.ts`:**
       ```typescript
       export interface DiatomicCurve {
         model: string;
         pair: string;
         distance: number[];
         energy: number[];
       }

       export interface DiatomicIndexResponse {
         data: Record<string, string[]>;
         meta: { count: number };
       }

       export interface DiatomicCurvesResponse {
         data: DiatomicCurve[];
         meta: { count: number };
       }
       ```

    2. **Add SWR hooks to `src/lib/api.ts`:**
       - `useDiatomicIndex()`: fetches `/api/v1/diatomics/index`, returns `{ index, isLoading, error }` where index is `Record<string, string[]>`.
       - `useDiatomicCurves(pair: string | null)`: fetches `/api/v1/diatomics/curves/${pair}` when pair is non-null, returns `{ curves, isLoading, error }`.

    3. **Create `src/components/PeriodicTable.tsx`:**
       - `'use client'` component.
       - Props: `availablePairs: Record<string, string[]>` (the diatomic index), `selectedElement1: string | null`, `selectedElement2: string | null`, `onSelectElement: (symbol: string) => void`.
       - Hardcode all 118 elements with CSS Grid positions (col 1-18, row 1-10 — standard periodic table layout including lanthanides row 9, actinides row 10).
       - Each element rendered as a small square button showing atomic symbol.
       - Three visual states:
         - **disabled** (grayed out, not clickable): no diatomic data exists for this element in any pair
         - **enabled** (normal): has data AND is a valid partner (when element1 selected, only elements that form a pair with element1 are enabled)
         - **selected** (highlighted border): currently selected as element1 or element2
       - When no element selected: enable all elements that appear in ANY pair key (split "H-H" → H appears).
       - When element1 selected: enable only elements where `${element1}-${element}` or `${element}-${element1}` exists as a key in availablePairs (pairs are alphabetically ordered, e.g., "Ac-H" not "H-Ac").
       - Use CSS Grid: `grid-template-columns: repeat(18, 1fr)` with 2px gap. Each element button: centered text, border-radius, hover effect, responsive sizing (min 28px, scales with container).
       - MUI-compatible colors: use `theme.palette` via `useTheme()` for element backgrounds, disabled opacity, selected border.
  </action>
